Learning-Based Computer Vision on Facial Expression Recognition

{{ site.description }}

This project focuses on the topic of facial expression recognition. Automatic detection on humans' facial expressions can be useful in real life but also challenging to be realized in daily application. In this project, we investigated on performances of some deep learning models on this task. Based on the results of these models on test data, we chose a best model and applied it to real-time detection.
